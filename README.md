# Description | sideProject-TodoList

the first todo list project done with basic JS DOM
######  Basic Function - CRUD
- 點擊＋或 ` enter ` 新增項目
- 點擊 ` deleteBtn ` 刪除該項目
- 點擊 ` editBtn ` 編輯該項目
- 判斷輸入重複內容/內容為空時跳出提醒視窗
- 點擊 clearAllHistory 清除所有紀錄
- 勾選後樣式更改

###### Extend Function
- 紀錄內容（包含勾選狀態），儲存至 ` localStorage `
